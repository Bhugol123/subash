ABOUT HTML


HTML, OR Hyper text Markup Language, is the programming language used to build the internet. It is the standrad language for web programming, along with CSS and JavaScript. This open-formate, text-based type code is extremely versatile, and can be used to embed additional programming language such as JavaScript. HTML is made up of different elements called tags that dicate the structure of the webpage that are then translated by a web browser into a user-friendly form.

MERN STAK
The MERN stack is a collection of technologies that help developers build robust and scalable web applications using JavaScript. The acronym “MERN” stands for MongoDB, Express, React, and Node. js, with each component playing a role in the development process.

DIGITAL MARKETING
Digital marketing, also called online marketing, is the promotion of brands to connect with potential customers using the internet and other forms of digital communication. This includes not only email, social media, and web-based advertising, but also text and multimedia messages as a marketing channel.

GRAPHIC DISIGN
Graphic design is a craft where professionals create visual content to communicate messages. By applying visual hierarchy and page layout techniques, designers use typography and pictures to meet users' specific needs and focus on the logic of displaying elements in interactive designs to optimize the user experience.
